# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-02-06

### Added
- Initial release of TTyper
- Solo practice mode with customizable text categories
- Multiplayer racing with real-time updates via Convex
- Lobby system with join codes
- Quick play matchmaking
- Player statistics and race results
- Multiple text categories: Quotes, Code, Random Words
- Keyboard-only navigation
- Terminal UI with themes
- GPL-3.0 License

### Technical
- Built with React and OpenTUI
- Real-time multiplayer using Convex subscriptions
- TypeScript for type safety
- Bun as runtime and package manager
